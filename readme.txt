A complete solution for managing Life in the UK citizenship tests.

== Description ==
Create and manage Life in the UK citizenship tests with features including:
- Question bank management
- Randomized test generation
- Timed tests
- Score tracking
- Customizable settings

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/life-in-uk-test` directory
2. Activate the plugin through the 'Plugins' screen
3. Configure settings under Life in UK → Settings
4. Add questions and create tests

== Frequently Asked Questions ==

= How do I create a test? =
1. Go to Life in UK → Question Bank and add questions
2. Navigate to Life in UK → Create Test
3. Enter a test name and click "Generate Random Test"

= Can I change the passing score? =
Yes, adjust the "Passing Percentage" in Life in UK → Settings

= How do I display tests? =
Use the shortcode [liuk_test id="X"] where X is your test ID
